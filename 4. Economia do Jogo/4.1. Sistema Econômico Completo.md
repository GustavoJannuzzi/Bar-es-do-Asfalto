### CONVERSÃO ENTRE TIPOS DE DINHEIRO

```
┌──────────────────────────────────────────────────────────────────────────┐

│ ──────────────────────────────────────────────────────────────────────   │

│ • Uso: Pode comprar qualquer coisa legalmente                           │

│ DINHEIRO SUJO (Crime, Não-Rastreável)                                   │
│ ──────────────────────────────────────────────────────────────────────   │

│ • Uso: Pagamentos informais, subornos, despesas pequenas                │
│ • Limite de uso direto: R$ 10.000 (acima disso levanta suspeita)        │
│ • Risco: Posse de grande quantidade é crime, precisa lavar              │
│ • Conversão: Precisa ser lavado para virar "limpo"                      │
├──────────────────────────────────────────────────────────────────────────┤
│ DINHEIRO EM TRÂNSITO (Sendo Lavado)                                     │
│ ──────────────────────────────────────────────────────────────────────   │
│ • Status: Temporariamente indisponível                                   │
│ • Processo: Passando por empresa de fachada, aguardando "limpeza"       │
│ • Tempo: 2-8 semanas dependendo do método                                │
│ • Risco: Se operação for descoberta durante processo, dinheiro se perde │
│ • Resultado: Após processo, vira "Limpo" (menos taxa de conversão)      │
└──────────────────────────────────────────────────────────────────────────┘
```

### REGRA DE CONVERSÃO

**Dinheiro Sujo → Dinheiro em Trânsito → Dinheiro Limpo**

- **Taxa básica de conversão:** 30% (você perde 30%, fica com 70%)
- **Taxa otimizada (bom contador):** 25%
- **Taxa premium (estrutura sofisticada):** 20%
- **Taxa arriscada (rápido e sujo):** 35-40%

**Exemplo:**
- Você tem R$ 100.000 sujos
- Usa empresa de fachada média (taxa 30%)
- Após 4 semanas, recebe R$ 70.000 limpos
- Perdeu R$ 30.000 no processo (custo da lavagem)
